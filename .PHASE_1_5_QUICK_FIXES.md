## Fix #2: Add Heart Animation Feedback ⏱️ 1 hour

**File:** `/app/inspiration/page.tsx`

**Problem:** Save heart button looks inactive when user saves. No visual feedback.

**Find (Line ~155):**
```typescript
function GalleryCard({ item, index, onSaveDream }) {
  // ...
  <button
    onClick={(e) => onSaveDream(item.id, e)}
    className="w-9 h-9 rounded-full bg-white/20 flex items-center..."
  >
    <Heart size={15} />
  </button>
}
```

**Replace with:**
```typescript
import { motion } from 'framer-motion'

function GalleryCard({ item, index, onSaveDream }) {
  const isSaved = isDreamSaved(item.id)  // Add this
  
  <motion.button
    onClick={(e) => onSaveDream(item.id, e)}
    whileHover={{ scale: 1.1 }}
    whileTap={{ scale: 0.95 }}
    className={`w-9 h-9 rounded-full flex items-center justify-center shrink-0 transition-colors ${
      isSaved
        ? 'bg-[#d4af37] text-white'
        : 'bg-white/20 text-white hover:bg-white/30'
    }`}
  >
    <motion.div
      animate={isSaved ? { rotate: [0, -15, 10, -5, 0], scale: [1, 1.2, 1] } : {}}
      transition={{ duration: 0.4 }}
    >
      <Heart 
        size={15} 
        className={isSaved ? 'fill-white' : ''}
      />
    </motion.div>
  </motion.button>
}
```

**Import `isDreamSaved` hook:**
```typescript
import { useDreams } from '@/contexts/dreams-context'

// Inside component:
const { isDreamSaved } = useDreams()
```

**Verify:** Click heart → color fills + animation plays
